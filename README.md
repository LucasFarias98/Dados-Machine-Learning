# Dados-Machine-Learning
Limpeza e Tratamento de Dados para Machine Learning

Um site interativo e visualmente impressionante para aprender técnicas de data cleaning
focadas em análise de churn de clientes. Tudo com design Cyberpunk Azul Neon e
explicações em dois níveis: técnico e simplicado (ELI12). O Que é Este Projeto?

Este é um site educacional completo em um único arquivo HTML5, ideal para quem
quer aprender como limpar, tratar e preparar dados para modelos de machine learning.
As anotações foram extraídas de um curso pro ssional, reformuladas para ser autorais, e
apresentadas em um formato super legal com:
✅
Design moderno e futurista com tema cyberpunk neon
✅
8 módulos de aprendizado bem estruturados
✅
Duas versões de cada tópico: explicação técnica completa + versão simpli cada (como
se explicasse para uma criança de 12 anos)
✅
Interface responsiva que funciona em celular, tablet e desktop
✅
Sem dependências externas (apenas Google Fonts)
✅
100% funcional — basta abrir no navegador!
Tópicos Abordados
1.  O que é Churn?
Entenda o conceito fundamental de taxa de cancelamento de clientes e por que isso
importa para os negócios.

2.  Fundamentos de Machine Learning
Como computadores aprendem com dados e por que isso é importante para prever
churn.

3.  Dados JSON & Normalização
Transforme arquivos JSON complexos em tabelas organizadas que você consegue
trabalhar.

4. ⚙ 
 Tipos de Dados & Casting
Garanta que seus dados têm o tipo correto para que o computador processe tudo
direito.

5.  Análise de Valores Únicos
Descubra padrões e anomalias nos seus dados.


6.  Remoção de Duplicatas
Livre-se de dados repetidos que deixam seu modelo confuso.

7. ⬜
 Tratamento de Valores Nulos
Aprenda estratégias para lidar com dados faltantes.
8.  Detecção de Outliers
Valores extremamente estranhos que podem prejudicar seu modelo.
